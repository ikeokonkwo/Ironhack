Keyboard Shortcuts

The following key combinations can be used in place of the menus. xScope must be the frontmost application for them to work correctly.

Keys Description
⌘1 Show or hide the Dimensions tool
⌘2 Show or hide the Rulers tool
⌘R Show Ruler properties
⌘E Toggles square ruler (extension) for all rulers
⌘I Inverts color for all rulers
⌘→ Rotate ruler 30º clockwise
⌘← Rotate ruler 30º counter-clockwise
⌘↓ Rotate ruler 1º clockwise
⌘↑ Rotate ruler 1º counter-clockwise
←, ↑, →, ↓ Arrow keys nudge ruler position by one pixel
⌥ ←,↑,→,↓ Option and arrow keys change the ruler's length by one pixel
⇧ ←, ↑, →, ↓ Shift and arrow keys move ruler position by 10 pixels
C Toggles selected ruler's calipers
P Toggles selected ruler's position indicators
E and L Toggles selected ruler's square (extension)
I Inverts selected ruler's color
M Toggles ruler alignment marks
2 Toggle alignment marks at 1/2 points
3 Toggle alignment marks at 1/3 points
4 Toggle alignment marks at 1/4 points
G Toggle alignment marks at Golden ratio
Home or End Rotates selected ruler by 1º (counter-clockwise or clockwise)
⇧Home or ⇧End Rotates selected ruler by ⅒º (counter-clockwise or clockwise)
Page Up or Down Rotates selected ruler by 5º (counter-clockwise or clockwise)
⌘3 Show or hide the Screens tool
←, ↑, →, ↓ Arrow keys nudge screen position by one point
⇧ ←, ↑, →, ↓ Shift and arrow keys move screen position by 10 point
⌘4 Show or hide the Loupe tool
⌘= Zoom in (plus sign to increase magnification)
⌘- Zoom out (minus sign to decrease magnification)
⌘L Lock position of loupe and continue display updates
⌘D Freeze display (no updates)
⇧⌘C Copy color at center of loupe to clipboard and palette
⌃⌘C Copy image of loupe to clipboard
←, ↑, →, ↓ Arrow keys nudge the loupe position by one pixel
⇧ ←, ↑, →, ↓ Shift and arrow keys move loupe position by 10 pixels
space The spacebar will clear the current loupe measurement
⌘5 Show or hide the Guides tool
⌘G Open guide wizard
⇧⌘H Create horizontal guide
⇧⌘V Create vertical guide
⌘ drag Holding down the command key will move all guides at once
⇧ drag Holding down the shift key will move in 10 pixel increments
⌥ drag The option key creates a copy of the guide and moves it
←, ↑, →, ↓ Arrow keys nudge current and linked guide positions by one pixel
⇧ ←, ↑, →, ↓ Shift and arrow keys move guide position by 10 pixels
H or ⇧H Select next or previous horizontal guide
V or ⇧V Select next or previous vertical guide
⌘6 Show or hide the Frames tool
⌘F Open frame wizard
⇧⌘F Create a frame
2 … 9 Toggle a grid on frame (e.g. 5 will toggle a 5 × 5 grid)
H Hide all grids on frame
⌘ drag Holding down the command key will move all frames at once
⇧ drag Holding down the shift key will move in 10 pixel increments
⌥ drag The option key creates a copy of the frame and moves it
←, ↑, →, ↓ Arrow keys nudge current and linked frame positions by one pixel
⌥ ←,↑,→,↓ Option and arrow keys change the frame's size by one pixel
⇧ ←, ↑, →, ↓ Shift and arrow keys move frame position by 10 pixels
F or ⇧F Select next or previous frame
⌘7 Show or hide the Crosshair tool
⌥⌘P Show mouse position with crosshair
⌘0 Set crosshair origin (zero point)
⌘W Close the Mirror, Text, Overlay, Screens or Loupe window
⌘T Toggle the toolbar window

The following key combinations are the default global hot keys. They will work in any application and can be changed in xScope Preferences

Keys Description
⌥⌘X Make xScope the frontmost application
⇧⌘5 Create a frame by shrinking around a selection
⇧⌘6 Create a frame from current Dimensions measurement
⇧⌘7 Take a screen shot with Dimensions, Overlay, Screens, Loupe or Frames
⇧⌘8 Create guides from current Dimensions measurement
⌃⌥⌘C Copy the current Loupe color to the clipboard and palette
